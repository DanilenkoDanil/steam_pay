Your API Documentation

api/just-pay/

Endpoint for making a payment.

Parameters:

  • login (string): Steam username.
  • amount (integer): Amount in ukz to transfer.

Request Headers:

  • Authorization: Token YOUR_AUTH_TOKEN

Example Request:

GET api/just-pay/?login=nickname&amount=7000 HTTP/1.1
Host: your-api-url.com
Authorization: Token YOUR_AUTH_TOKEN
